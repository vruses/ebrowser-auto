# ebrowser-auto

`ebrowser-auto` 是一个调用api实现批量自动化浏览器操作的nodeJs脚本项目，包括批量打开关闭浏览器。

## 快速开始

1. 克隆仓库：

  ```bash
  git clone https://github.com/vruses/ebrowser-auto.git
  ```

2. 安装依赖：

  ```bash
  yarn
  ```

3. 运行示例脚本：

  ```bash
  yarn start
  ```

4. 构建可执行文件：

  ```bash
  yarn build
  ```
